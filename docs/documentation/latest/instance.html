    • instance

    This namespace lists utility functions attached to an instance of Rac used to produce drawable and other objects, and to access ready-build convenience objects like instance.Angle#north or instance.Point#zero.

    Drawable and related objects require a reference to a rac instance in order to perform drawing operations. These functions build new objects using the calling Rac instance, and contain ready-made convenience objects that are also setup with the same Rac instance.
