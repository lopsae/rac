    Members and methods attached to the rac.Segment function.

    The function contains ready-made convenience Segment objects for usual values, all setup with the owning Rac instance.

    Example
    let rac = new Rac()
rac.Segment.zero // ready-made zero segment
rac.Segment.zero.rac === rac // true
