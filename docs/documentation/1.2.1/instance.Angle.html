    Members and methods attached to the rac.Angle function.

    The function contains ready-made convenience Angle objects for usual values, all setup with the owning Rac instance.

    Example
    let rac = new Rac()
rac.Angle.quarter // ready-made quarter angle
rac.Angle.quarter.rac === rac // true
