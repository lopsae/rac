    # | source: src/style/Fill.js, line 38

    Fill color for drawing.

    Can be used as fill.apply() to apply the fill settings globally, or as the parameter of drawable.draw(fill) to apply the fill only for that call.

    When color is null a no-fill setting is applied.

    instance.Fill

    Instances of Rac contain a convenience rac.Fill function to create Fill objects with fewer parameters. This function also contains ready-made convenience objects, like rac.Fill.none, listed under instance.Fill.

    Example
    let rac = new Rac()
let color = rac.Color(0.2, 0.4, 0.6)
// new instance with constructor
let fill = new Rac.Fill(rac, color)
// or convenience function
let otherFill = rac.Fill(color)

    # | source: src/style/Fill.js, line 77

    Returns a Fill derived from something.

    • When something is an instance of Fill, returns that same object.
    • When something is an instance of Color, returns a new Fill using something as color.
    • When something is an instance of Stroke, returns a new Fill using stroke.color.
    • Otherwise an error is thrown.
