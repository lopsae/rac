    Rac. Arc

    Arc of a circle from a start angle to an end angle.

    Arcs that have the same start and end angles are considered a complete circle.

    clampToInsets(angle, startInsetopt, endInsetopt) → {Rac.Angle}

    Returns the given angle clamped to the range:

    [start + startInset, end - endInset]

    where the addition happens towards the arc's orientation, and the substraction against.

    When angle is outside the range, returns whichever range limit is closer.

    When the given insets are larger that the arc distance between start and end the range for the clamp becomes imposible to fulfill. In this case the returned value will be the centered between the range limits and still clampled to [start, end].

    Returns a new Arc with the given length as the length of the part of the circumference it represents.
