segmentTangentToArc(arc, clockwiseopt) → {Rac.Segment}

Returns a new Segment that is tangent to arc in the clockwise orientation from the ray formed by this and arc.center. Returns null if this is inside arc and thus no tangent segment is possible. The returned Segment starts at this and ends at the contact point with arc which is considered as a complete circle.
