    • Rac. Control

    Parent class for all controls for manipulating a value with the pointer. Represents a control with a value, value-range, limits, markers, and drawing style. By default the control returns a value in the range [0,1] coresponding to the location of the control center in relation to the anchor shape. The value-range is defined by startValue and endValue.
